# 👟 StepStyle - Online Footwear Shop

A full-featured online footwear shop built with Python Flask.

## Features
- 🛍️ Product catalog with categories
- 🔍 Search & filter products
- ⭐ User rating system
- 🛒 Shopping cart
- 💳 Checkout with Cash on Delivery
- 👤 User profile page
- ⚙️ Admin dashboard

## Tech Stack
- Python Flask
- SQLite Database
- HTML, CSS, Jinja2

## Run Locally
```bash
pip install -r requirements.txt
python app.py
```
